About NotEBook

Throughout our academic careers, we have noticed that technology has increasingly more integral in providing a complete educational experience. Subsequently, we set out to develop a software application which complements the integration of technology into students' academic structure. NotEBook accomplishes this ideology through providing an intuitive user interface accompanied by a convenient set of note-taking tools and apparatuses.

Features

Desktop/Web Access

We offer the mobility of supporting seamless desktop and website accessibility

Local and Cloud Storage

All of your files will save to a cloud storage unit when connected to the internet, allowing for synchronized data storage accross our desktop and website platform. We also offer the option of storing your files locally for your convenience!

Text Editor

Some quick example text to build on the card title and make up the bulk of the card's content.

Calendar/ Planner

Plan your schedule with ease with our intuitive event planner!

Drawing Canvas

If it's more convenient for you to write out your notes by hand, we offer the ability to write your notes on a digital ink canvas!

Calculator

If you need to crunch some numbers really quickly, use our intuitive built-in calculator!
